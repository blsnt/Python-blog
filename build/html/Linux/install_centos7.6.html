1. 安装VMware虚拟化软件¶

软件百度云分享链接 提取码为:1y2s 找到vmware双击安装，一直下一步就完事了，里面有激活码 找到文件新建虚拟机 默认自定义就行 下一步 选择稍后安装操作系统 选择Linux 版本：centos64位 定义虚拟机的名称和安装位置。可以默认 选择CPU核数 根据你的系统配置来调节内存。我这里默认1G 选择网络NAT地址转换模式 默认即可 默认即可 默认即可 选择磁盘空间大小 默认即可 完成

2.2. 安装操作系统¶

点击编辑虚拟机设置 选择镜像 开启虚拟机 鼠标点击进去按Tab键输入 biosdevname=0 net.ifnames=0 直接回车 更改网卡为eth0 选择安装语言 选择时间 选择上海 键盘默认即可 选择系统需要安装的语言。我这里只选择英语，你也可以添加中文 安装源默认即可 选择软件安装 配置网络 选择开机自动连接 关闭安全(工作中建议打开) 磁盘格式化并分区 自定义分区 选择标准分区 swap在工作中内存大于8g就给8g，小于8g就给内存的1.5倍 根分配所有 设置root密码(工作中必须8位以上，数字+字母+特殊字符) 等待安装完成 最后点击reboot会重启。进入操作系统 输入用户名：root 密码：666666 进入系统 虚拟机网络配置 可以使用ping命令检测是否可以上网 如果不可以上网可以使用systemctl restart network重启网络 安装系统后：nmtui 命令可以使用图形化配置网卡和主机名

3. 安装xshell远程连接服务器¶

软件百度云分享链接 提取码为:1y2s 找到xshell直接下一步下一步就完事了 安装完成新建一个会话 连接配置 输入用户名和密码 点击连接 接受并保存 已经连接上服务器 xshell属性配置 文件->属性->终端 基础命令 安装下载工具wget 更改yum源 阿里云源下载地址 epel源同上

$ wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo
$ wget -O /etc/yum.repos.d/epel.repo http://mirrors.aliyun.com/repo/epel-7.repo

将系统软件更新到最新

$ yum update -y

CentOS6和CentOS7都要安装的运维常用基础工具包

$ yum -y install tree nmap dos2unix lrzsz nc lsof wget tcpdump htop iftop iotop sysstat nethogs

CentOS7要安装的运维常用基础工具包

$ yum -y install psmisc net-tools bash-completion vim-enhanced -y

安装的时候选包，选了三个，组包：里面有一堆软件。如果没有安装请用以下方式安装

$ yum groups mark convert
$ yum groupslist 查看所有包组名称

安装包组名称

$ yum groupinstall "Development Tools" -y
